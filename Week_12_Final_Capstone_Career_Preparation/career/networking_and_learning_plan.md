# 90-day networking and continuing-learning plan

## Weeks 1-4: portfolio and fundamentals

- Publish Week 9-12 repositories with clear READMEs, screenshots, results, and limitations.
- Practice SQL joins, grouping, window functions, subqueries, and case studies.
- Review statistics, classification metrics, preprocessing, and model interpretation.
- Apply to 5-8 relevant internships or entry-level roles each week.

## Weeks 5-8: deployment and communication

- Deploy the Streamlit capstone and add its link to GitHub and LinkedIn.
- Record a 2-minute project explanation and improve it from feedback.
- Build one dashboard project using Power BI or Tableau.
- Conduct two mock interviews each week.

## Weeks 9-12: specialization and interviews

- Learn time-series basics or NLP after strengthening SQL and analytics.
- Add experiment design, A/B testing, calibration, and monitoring concepts.
- Reach out to alumni, analysts, recruiters, and local tech communities.
- Track application stage, follow-up date, outcome, and skill gaps.
